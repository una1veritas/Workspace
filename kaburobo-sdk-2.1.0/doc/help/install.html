インストール方法

カブロボ SDK は JDK 1.6 を利用して作られています。JDK 1.5 でも互換動作します。
JDK が入っていない、もしくは 1.4 以前のものしかない場合は、java.sun.com のサイトからダウンロードしてインストールしてください。

まず、カブロボ SDK の zip ファイル (kaburobo-*.*.*.zip: *.*.* はバージョン) をダウンロードし、適当なディレクトリに展開します。
このディレクトリを以後「カブロボホームディレクトリ」と呼びます。

次に、環境変数に KABUROBO_HOME を追加します。
変数 KABUROBO_HOME に、値を カブロボホームディレクトリの絶対パスにします。
環境変数の設定につきましてはお使いのOSによって異なる為、OSに合わせて適宜設定して下さい。

これで初期設定は終了です。

設定の確認を行なうために、コマンドプロンプトを開き、次のコマンドを実行します。

OS別コマンド表
OSコマンド
Windows %KABUROBO_HOME%\kaburobo
Unix $KABUROBO_HOME/kaburobo.sh

ただし、Windowsで環境変数 KABUROBO_HOME のパスに空白を含む場合（例: C:\Program Files\Kaburobo\SDK など）は、 下記のようにダブルクオーテーション「"」で囲んで実行してください。

OS別コマンド表
OSコマンド
Windows "%KABUROBO_HOME%\kaburobo"

次のようなメッセージが表示されるか確認してください。
