XRogue: 運命の洞窟への遠征
--------------------------

はじめに:

Rogueは、カリフォルニア大学バークレー校で紹介された、スクリーン志向のファン
タジーゲームです。このゲームは、プレーヤが、コンピュータが生成したダンジョン
を探検し、26種類の怪物に出会います。巻物、ポーション、指輪、杖や棒、鎧、そし
て武器が、プレーヤが、怪物と戦ったりスコアの基準となる金塊を手に入れる手助け
となります。

このガイドに記述されているバージョンのローグは、多くの新しい能力と共に200種
類以上の怪物を含むように拡張され、名前もXRogueに換えられました。多くの怪物は
賢く、プレーヤのように罠を避けなければならなかったり、いつ戦うかいつ逃げるか
を考えます。また、前のバージョンには無かったたくさんの新しいコマンドがありま
す。

このゲームには、怪物、呪文、武器、鎧、ポーション、そしてあなたがクエストの間
に発見する魔法のアイテムがたくさんあります。洞窟の地図はゲームごとに変わりま
すし、多くの魔法のアイテムに、プレーヤの目を見えなくすると言ったような固有の
性質はあるのですが、アイテムの見た目は個々のゲームで変わります。例えば、赤い
ポーションはあるゲームの間は同じ反応を引き起こすでしょうが、新しくゲームを始
めると完全に異なった反応のポーションであるかも知れないのです。

少しの食料、鎧、そして武器を持って洞窟に入り、プレーヤは、いつ戦いいつ逃げ、
そして洞窟で見つけたあらゆる魔法のアイテムの最善の使い道を考えるという、良い
戦略を考え出さなければなりません。プレーヤが興味を持つことは、深い洞窟の中に
横たわっていると噂のあるいくつかのユニークな魔法のアーティファクトの内の一つ
を持ち帰ると言うクエストにあります。これらのアーティファクトを地上に持ち帰る
ことができれば、偉大なる名誉が得られるでしょう。

しかしながら、アーティファクトを見つけた後も、プレーヤが望むならば、悪魔、悪
魔の王子、またはカロンや船頭といっしょに、洞窟の奥深くまで理知に合わせてクエ
ストを続けていくことも可能です。そのような生き物たちをうち破ることにより、プ
レーヤはXRogueのスコアの基本となる経験値を多く得ることができるでしょう。運命
の洞窟から生きて帰ることは非常に難しいことです。このゲームにうち勝ったプレー
ヤはほとんどいません。

キャラクタクラス:

洞窟の中にプレーヤを配置する前に、どんなタイプのキャラクタでありたいかを選択
することを要求されます: Fighter(ファイター、戦士)、Paladin(聖騎士)、
Ranger(レンジャー)、Magic-User(魔法使い)、Cleric(牧師)、Thief(盗賊)、
Assassin(刺客)、Druid(ドルイド僧)、またはMonk(修道士)。

    Fighter(ファイター)

Fighter(ファイター)は怪物と戦って勝つには、最も勝ち目があります。高い経験値
を持つ場合、Fighter(ファイター)は１ターンに複数回の攻撃を仕掛けることもでき
ます。Strength(力)はFighter(ファイター)の一番重要な属性です。

    Magic-User(魔法使い)

Magic-User(魔法使い)は呪文をかけることができます。Intelligence(知性)が一番重
要な属性です。Magic-User(魔法使い)がかけることのできる呪文の数は、経験値や
Intelligence(知性)が増すと増加します。洞窟内において、呪文をかける能力は、あ
らゆるアイテムと同一視されます。16の呪文。

    Cleric(牧師)

Cleric(牧師)は戦闘中、アシストのために祈りを捧げることができます。Wisdom(知
恵)が一番重要な属性です。Cleric(牧師)に与えられる祈りの数は、経験値や
Wisdom(知恵)が増すと増加します。Clerics(牧師)はゾンビやグール(食屍鬼)などの
不死の怪物との戦いを避けることができます。Cleric(牧師)は、不死の怪物に対し
て、奴らを完全に撲滅するほど非常に強力です。16人の祈り。

    Paladin(聖騎士)

Paladin(パラディン)は、Cleric(牧師)とFighter(ファイター)の中間的な、神聖なる
戦士のタイプです。Cleric(牧師)のように祈り不死の怪物に影響を与えることもでき
ますし、Fighter(ファイター)のように戦うこともできますが、それらより効果は劣
ります。彼は総て正義の側にいて、最初に彼を攻撃しなかった怪物をけっして攻撃し
たりはしないでしょう。もし彼がそのような怪物を殺すならば、彼はますます心配に
感じ始めます。もし彼が非常に多くそのような怪物を殺すなら、総てのCleric(牧師)
の能力が失われ、単なるFighter(ファイター)に落ちぶれるでしょう。Charisma(カリ
スマ)はWisdom(知恵)の次に大事な属性です。

    Ranger(レンジャー)

Ranger(レンジャー)は、Magic-User(魔法使い)とFighter(ファイター)の中間的な、
神秘的な戦士のタイプです。Paladin(パラディン)のように、彼は総て正義の側にお
り、最初に彼を攻撃しなかった怪物をけっして攻撃しないでしょう。Ranger(レン
ジャー)はMagic-User(魔法使い)のように呪文をかけることもできますし、
Fighter(ファイター)のように戦う事もできますが、それらより効果は劣ります。
Charisma(カリスマ)はIntelligence(知性)の次に大事な属性です。

    Thief(盗賊)

Thief(盗賊)は、特に器用で、怪物からアイテムを奪う(盗む)ために罠を仕掛けたり
する素晴らしいスキルがあります。Thieves(盗賊)は、洞窟の個々の階にある総ての
近海や隠された罠を総て察知する能力を持っています。天賦のDexterous(器用さ)で
彼らは非常に静かに動作が行えるので、他のどのキャラクタタイプよりもずっと、寝
ている怪物を起こすこと無しに行動できるのです。もし、Thief(盗賊)が何とかして
怪物を起こすことなく忍び寄ることができたなら、怪物を後ろから仕留めることだっ
てできるかもしれないのです。後ろから刺すことによって与えるダメージで、経験レ
ベルが非常に増加します。Dexterity(器用さ)が一番重要な属性です。

    Assassin(刺客)

Assassin(刺客)は不意打ちで怪物を仕留める芸術的な訓練を受けた人物です。彼は多
少、盗賊の能力も併せ持ちますが、罠を感じ取ったり背後から忍び寄って仕留めたり
することはできません。その代わり、Assassin(刺客)は致命的な一撃で相手を完全に
仕留められるチャンスを持っています。彼は、洞窟で見つかる毒が何であるか認識
し、それを武器に用い、それによって次の攻撃を特別致命的な物にすることができま
す。Dexterity(器用さ)が一番重要な属性です。

    Druid(ドルイド僧)

Druid(ドルイド僧)は、Cleric(牧師)とMagic-User(魔法使い)の中間的な、魔法の戦
士というタイプです。Druid(ドルイド僧)は、自分自身の持ついくつかを加えて、呪
文と聖歌の両方を唱えることができます。Druid(ドルイド僧)が歌うことのできる聖
歌の数は、Wisdom(知恵)の経験値が増加するに従って増えます。Wisdom(知恵)が一番
重要な属性です。16の聖歌。

    Monk(修道士)

Monk(修道士)は、武道で鍛えられています。彼は何の鎧も着ませんし、武器も必要と
はしません(別に使用を禁じられている訳ではありませんが)。Monk(修道士)は経験値
が増すにつれて、攻撃者に対して、自然と防御力が増し、身をかわす能力もついてき
ます。Monk(修道士)はDruid(ドルイド僧)とFighter(ファイター)の中間なので、彼は
聖歌を歌うこともFighter(ファイター)のように戦うこともできますが、両方とも彼
らより効果は劣ります。Constitution(気質)が一番、Wisdom(知恵)が二番目に重要な
属性です。

キャラクタの属性:

Strength(力) - 力に対する負荷のための最も重要な属性。

Intelligence(知性) - 呪文を唱えるための最も重要な属性。

Wisdom(知恵) - 祈りや聖歌のための最も重要な属性。

Dexterity(器用さ) - 誰にも気付かれずに行動できるための最も重要な属性。

Charisma(カリスマ) - 善意ための最も重要な属性。取引をする際、高いCharisma(カ
                     リスマ)性は物の値段に影響します。

Constitution(気質) - 健康ための最も重要な属性。高いConstitution(気質)は、経
                     験レベルが上がる時、どのくらいHit Points(ヒットポイン
                     ト)が増加するかに影響します。

注意: Ranger(レンジャー)、Paladin(聖騎士)、そしてMonk(修道士)は、いくつか経
験レベルが増えるまで、彼らの「特別な」魔法の能力を使えません。

経験レベル:

キャラクタの経験値は、たいていは怪物を倒すことで増加します。他の行動(怪物か
らアイテムを盗むとか、背後から刺すとか、怪物を追い返すとか)は、更に経験値を
増加させます。個々のキャラクタタイプはそれぞれ異なる割合で、経験値が増加した
り、経験レベルが上がったりします。経験レベルが上がるにつれて、キャラクタの
ヒットポイントは増加しますが、殺すまでに何発の「ヒット」を与えたかによって変
わってきます。

キャラクタに属性ポイントを割り当てる:

プレーヤは最初に、ゲームで選んだキャラクタに、属性ポイント75を分配する所から
始めます。あなたが属性ポイントを分配するように言われる時、スクリーンにはその
特定の属性に対する、設定可能な値の最大と最小が表示されています。プレーヤは、
Backspaceキー(Ctrl-H)で戻って前の値を変えられますし、Escape(ESC)キーで、配布
された属性ポイントの残りの数を、残りの総ての属性値を可能な限り大きな値にセッ
トしたりできます。

スクリーン

通常のプレイの間、スクリーンは3つの別々のセクションから構成されています: 最
上行、一番下の2行、そして残りの真ん中の部分です。最上行はゲームの間どんなこ
とが起こったのかを報告し、真ん中の部分は洞窟を表示し、下の2行はプレーヤの現
在の状態を表示します。

スクロールを見つけたとか、怪物を攻撃したとか攻撃されたとか、プレーヤに起こっ
たことは何でも、スクリーンの最上行に簡単に報告が表示されます。最上行に
「More」と言う単語が表示されている時は、続きを見るためにはスペースキーを押さ
なければならないことを意味しています。

以下のアイテムは、洞窟の中で見つかるかもしれません。いくつかのアイテムは、あ
なたのキャラクタがこれらを理解しているかどうかによって、複数の意味を持つこと
があります。

|  部屋の壁。
-  部屋の壁。
*  金塊。
%  他の階への階段。
+  部屋のドア。
.  部屋の床。
#  通路の床。
   硬い岩(スペースで示される)。
^  Trading Post(交易場)への入り口。
@  プレーヤ。
_  目に見えない状態のプレーヤ。
:  いくらかの食料。
!  ポーションの入ったフラスコ。
?  密封されたスクロール。
=  指輪。
)  武器。
]  いくらかの鎧。
;  種々雑多な魔法のアイテム。
,  アーティファクト。
/  杖や棒。
>  次の階へ落とされる罠のドア。
{  弓矢の罠。
$  催眠ガスの罠。
}  ネズミ捕り。
~  テレポートする罠。
`  毒矢の罠。
"  ゆらめく魔法のプール。
'  迷路の入り口。
$  何かの魔法のアイテム。(魔法発見の間)
>  祝福された魔法のアイテム。(魔法発見の間)
<  のろわれた魔法のアイテム。(魔法発見の間)

怪物はアルファベットで表現されます。注意して欲しいのは、あなたが洞窟のどの階
にいるかによって、総ての文字が複数の別の怪物を意味することです。プレーヤは識
別コマンド(「/」)か説明コマンド(「=」)で今いる怪物が何なのか識別することも出
来ます。

スクリーンの下の2行は、プレーヤの状態を表示します。上の方の行は、プレーヤの
属性を表します:

Intelligence(知性)、Strength(力)、Wisdom(知恵)、Dexterity(器用さ)、
Charisma(カリスマ)、そしてConstitution(気質)、総て最大値は50ポイントですが、
指輪の力を得るともっと高い値にすることもできます。負荷は、プレーヤが最大限運
べる：現在運んでいるの測度です。あなたの最大負荷によって、あなたがより多くの
ものを運ぶに従い、より多くの食料を必要となります。Strength(力)の属性は、その
人の負荷を強くします。

プレーヤの現在のヒットポイントは(Hp)で表示され、現在の最大ヒットポイントが
カッコ内に示されます。ヒットポイントはプレーヤの生存性を表します。プレーヤが
休憩やポーション、呪文などで回復していくと、プレーヤの現在のヒットポイントが
現在の最大値に到達するまで増えて行きます。プレーヤの経験レベルが上がるたび
に、この最大値は増えていきます。もし、プレーヤの現在のヒットポイントが0に
なってしまったら、プレーヤは「代謝障害」になります。

プレーヤのアーマークラスは(Ac)で表示されます。この数値は、鎧や外套、プレー
ヤが身に着けている鎧や外套、指輪などで得られる防御力を示しています。これは、
Dexterity(器用さ)の高い低いにも影響されます。何の鎧も身に着けていないと、
アーマークラスは10になります(Monk(修道士)を除きます)。アーマークラスの数値が
低いと、防御力は高くなります。

プレーヤの現在の経験レベルは(Exp)で表示され、次に経験ポイントが続きます。次
の経験レベルになるとヒットポイントが増加し、可能ならば他の能力も増えます。例
えば、Magic-user(魔法使い)の新しい呪文や、Cleric(牧師)の新しい祈り、Druid(ド
ルイド僧)の新しい聖歌などです。それぞれのキャラクタに付き、26の経験レベルが
あります。

コマンド:

プレーヤは、1文字入力するだけでほとんどのコマンドを実行する事ができます。し
かしながら、いくつかのコマンドは方向が必要ですので、その場合は続けて方向の文
字を入力します。多くのコマンドは、数字に続けて入力する事もでき、その場合は何
回コマンドを実行しなければならないかを示します。

プレーヤが荷物に入っているアイテムを参照するようなコマンド(例えばスクロール
を読むなど)を実行する際、どのアイテムにするか決めるためのプロンプトを出しま
す。プレーヤはアイテムに付けられた文字を入力する事ができます。「*」を入力す
ると、条件に合うアイテムのリストを表示します。

基本的なゲーム・コマンドのリスト:

?  「?」に続けてコマンドを入力すると、コマンドの簡単な説明が表示されます。コ
   マンド「?*」は、総てのコマンドの説明になります。「?@」は、あなたの出くわ
   すもの(岩、森、など)の情報をもたらします。
/  「/」に続けてシンボルを入力すると、そのシンボルを特定します。
=  説明。「=」記号を入力した後、プレーヤは移動キーを使って、現在の階のどこで
   もカーソルで指し示せます。プレーヤが選択された場所を普通に見られる限り、
   ゲームはいったい何がそこにあるかについて識別してくれます。
h  左に1コマ移動します。
j  下に1コマ移動します。
k  上に1コマ移動します。
l  右に1コマ移動します。
y  左上に1コマ移動します。
u  右上に1コマ移動します。
b  左下に1コマ移動します。
n  右下に1コマ移動します。
H  何か意味のあるものに出会うまで、左に走ります。
J  何か意味のあるものに出会うまで、下に走ります。
K  何か意味のあるものに出会うまで、上に走ります。
L  何か意味のあるものに出会うまで、右に走ります。
Y  何か意味のあるものに出会うまで、左上に走ります。
U  何か意味のあるものに出会うまで、右上に走ります。
B  何か意味のあるものに出会うまで、左下に走ります。
N  何か意味のあるものに出会うまで、右下に走ります。

>  階段を下りて次の階に行きます。または、ワームホールの罠に落ちている時、外
   の領域に入ります(これができるためには「飛べ」なければなりませんが)。
<  階段を下りて次の階に行きます。または、ワームホールの罠に落ちている時、外
   の領域に入ります(これができるためには「飛べ」なければなりませんが)。

*  プレーヤの荷物に入っている金塊を数えます。
!  OSのシェルに抜けます。
$  Trading Post(交易場)で、アイテムの値段を聞きます。
#  Trading Post(交易場)で、アイテムを買います。
%  Trading Post(交易場)で、アイテムを売ります。
.  このコマンド(ピリオド)は、1ターンの間プレーヤを休ませます。
^  このコマンドは、罠を仕掛けます。Thieves(盗賊)とAssassins(刺客)だけしか使
   えません。このコマンドがうまくいけば、ゲームがどんなタイプの罠を仕掛ける
   のか聞いてきて、プレーヤの立っている場所に仕掛けます。
a  アンデッドに攻撃します。このコマンドはClerics(牧師)とPaladins(聖騎士)だけ
   しか使えません。コマンドに続けて、方向を表す文字の入力が必要です。
A  あなたのクエストのアイテムを選びます(ゲームの開始時だけです!)。
c  このコマンドはDruids(ドルイド僧)とMonks(修道士)だけしか使えません。歌う事
   のできる聖歌などのリストを表示します。プレーヤは表示された聖歌の内の1つを
   選べます。そして唱えるだけのエネルギーがプレーヤに十分あるなら、それを
   「聖歌を歌い」ます。祈りが高度であればあるほど、エネルギーもたくさん消費
   します。
C  このコマンドはMagic-Users(魔法使い)とRangers(レンジャー)だけしか使えませ
   ん。唱える事のできる呪文のリストを表示します。プレーヤは表示された呪文の
   内の1つを選べます。そして唱えるだけのエネルギーがプレーヤに十分あるなら、
   それを「唱え」ます。呪文が高度であればあるほど、エネルギーもたくさん消費
   します。
d  プレーヤの荷物からアイテムを捨てます。
D  魔法のプールに何かを浸します。
e  プレーヤの荷物にある食料を食べます。
f  このコマンドに続けて方向コマンドを入力すると、プレーヤは何か意味のあるも
   のに隣接するまで移動し続けます。
F  怪物を怖がらせます。総てのキャラクタが使える訳ではありません。このコマン
   ドは、10回辺りで効果を失います。
g  怪物に食料であるスライム・モールドを、タダであげるか交換するかします。
G  このコマンドはThieves(盗賊)とAssassins(刺客)だけしか使えません。現在の階
   にある総ての金塊を表示します。
i  プレーヤの荷物の目録を表示します。
I  このコマンドは、プレーヤの荷物の中のある1つのアイテムを選ぶためにプロンプ
   トを出し、そのアイテムの情報を表示します。
m  プレーヤがこのコマンドを入力すると、アイテムに1行の名前を付ける事ができま
   す。
o  このコマンドで、ゲームに設定可能なオプションが総て表示されます。プレーヤ
   はそれを調べたり、いくつかを変更したりできます。
O  あなたの現在のキャラクタタイプとクエストのアイテムを表示します。
p  このコマンドはClerics(牧師)とPaladins(聖騎士)だけしか使えません。使う事の
   できる祈りのリストを表示します。プレーヤは表示された祈りの内の1つを選べま
   す。そして唱えるだけのエネルギーがプレーヤに十分あるなら、それを「唱え」
   ます。祈りが複雑であればあるほど、エネルギーもたくさん消費します。
P  現在プレーヤの下に落ちているアイテムを拾います。
q  プレーヤの荷物にあるポーションを飲みます。
Q  ゲームをセーブせずに終了します。
r  プレーヤの荷物にあるスクロールを読みます。
s  プレーヤの回りで、隠されたドアや罠を探します。
S  後でゲームが再開できるようにセーブします。
t  このコマンドはプレーヤの荷物から1つの物体取り出します。そしてそれを、プ
   レーヤは指定の方向に投げる事ができます。
T  プレーヤの身につけているものら何でも脱ぎ/外します。
v  現在のXRogueのバージョンを表示します。
w  プレーヤの荷物にある武器を装備します。
W  プレーヤの荷物に入っている鎧や指輪、様々な魔法の品を身につけます。プレー
   ヤは最大8つの指輪をする事ができます。
X  このコマンドはThieves(盗賊)だけしか使えません。現在の階にある隠された罠を
   総て表示します。
z  このコマンドはプレーヤの荷物に入っている杖や棒を振りかざすためのもので、
   方向を指定するプロンプトを表示します。
+  フォーチュンクッキー! (注意: もしあなたがXRogueをモデム経由でプレーしてい
   るなら、3回続けて「+」を押すとモデムが「コマンド」モードに入ってしまいま
   す)。

Escape  Escapeキーを押すと、現在のコマンドをキャンセルします。
Ctrl-B  現在のあなたのスコアをチェックします。スコアは経験値と金塊に基づいて
        います。しかしながら、金塊はそんなに重要な訳ではなく、もしプレーヤが
        死んでしまうと10%払わなければなりません。
Ctrl-E  現在のあなたの満腹/空腹度をチェックします。このコマンドは、あなたが
        胃の中にどれだけの食べ物が残っているのか知りたい時に使います。満腹時
        はおよそ2000(2100)です。あなたがゲームをプレーして行くにつれてその値
        は下がっていき、およそ200(2100)で空腹となります。レベル2000(2100)以
        上になるとキャラクタはうんざりしてきます。レベルが200(2100)を下回る
        とキャラクタは空腹になり、だんだん弱ってきて、最後には気絶して倒れて
        しまいます。レベル2000(2100)が、ほとんどのキャラクタが食べる事のでき
        る、満腹で満足した胃の状態です!
Ctrl-L  スクリーンを再表示します。
Ctrl-N  プレーヤがこのコマンドを入力すると、怪物やプレーヤの荷物にあるアイテ
        ムに1行の名前を付ける事ができます。怪物に名前を付けるには、カーソル
        を適当な怪物の上に置いて、名前を付け直してください。
Ctrl-O  現在のプレーヤの「影響」(例えば、遅い、局面、混乱している、特別な視
        力、飛んでいる、踊っている、など)を表示します。
Ctrl-R  スクリーン最上行に出た最後のメッセージを再表示します。
Ctrl-T  このコマンドはThieves(盗賊)とAssassins(刺客)だけしか使えません。続け
        て方向を示す文字を入力しなければなりません。怪物がプレーヤの隣の指定
        方向にいたならば、怪物の荷物から1つのアイテムを盗む事ができます。成
        功したならば怪物は何も気付きませんが、もし失敗したなら怪物が突然起き
        出して攻撃してくるという可能性があります。
Ctrl-U  プレーヤの荷物にある魔法のアイテムを使います。

ここには明白な攻撃のコマンドがありません。プレーヤが怪物と戦闘をしたいなら、
プレーヤは単純に怪物の立っている場所に乗るように移動すればよいのです。プレー
ヤが装備しているものは何でも、プレーヤの武器として扱われます。

プレーヤがアイテムを横切ると、ゲームは自動的にそれを拾い上げ、プレーヤの荷物
の中にしまいます。もし荷物の中に入れる場所がない場合、そのアイテムは床の上に
残されたままになります。「pickup」オプションを「NO」にしてあると、「P」コマ
ンドを使った時だけアイテムを拾うようになります。

帳簿命令(持ち物リストを見るなど)以外の全ての行動は、時間が掛かります。掛かる
時間の量はコマンドによって異なります。例えば、武器を振り回すのは単に動くだけ
よりも時間がかかります。という訳で、プレーヤが1回攻撃をする間に、怪物は何歩
か移動する事ができるのです。武器を振り回すのにかかる時間は武器の大きさによっ
て異なります。そして、単に1歩移動するのにかかる時間は鎧の種類とプレーヤの力
のレベルによって異なります。プレーヤが「飛んでいる」時、移動は常に速くなりま
す。

行動も時間がかかります。そして、行動が中断させられる事もあります。例えばプ
レーヤが魔法をかけていて、これをかけ終わる前に攻撃されると、呪文の効果は失わ
れてしまいます。同様に、食べようとしている時に攻撃されると、窒息してしまうか
もしれません。同じ規則は，怪物にも当てはまります。

洞窟の部屋のいくつかは、天然の光源があります。それ以外の多くの部屋や通路で
は、プレーヤは自分の回りの1歩分の範囲しか見る事はできません。暗い部屋は、魔
法のランプか、ファイヤビートルと他の怪物たちによって照らされます。

プレーヤは一度に1つの武器だけを装備する事ができます。プレーヤが怪物に攻撃を
仕掛ける時、与えるダメージの量は装備している個々の武器に依存します。発射する
たぐいの武器、例えばクロスボウやショートボウなどを使うには、プレーヤはまず弓
を装備し、怪物にボルトや弓矢を射ねばなりません。

武器は呪われていたり祝福されていたりします。それが、あなたが怪物を攻撃できる
可能性や、怪物に与えられるダメージに影響してきます。もしプレーヤが自分の使っ
ている武器を識別できたなら、「攻撃」と「ダメージ」のボーナスポイントが、持ち
物リストの武器名の前にこの順で表示されます。プラスのボーナスポイントはそれが
祝福された武器である事を示し、マイナスの場合は呪われているか見当外れの武器で
ある事を示しています。呪われた武器は、呪い解除のスクロールを読むか魔法をかけ
るかしないと、装備を解除する事ができません。

プレーヤが鎧一式の識別を終えると、持ち物リストの鎧名の前に防御のボーナスポイ
ントが表示されます。もしボーナスがプラスであればそれが祝福されたものであり、
もしマイナスであればその鎧はたいてい呪われた物でしょう。呪われた鎧一式は、呪
い解除のスクロールを読むか魔法をかけるかしないと、脱ぐ事ができません。

いくつかの怪物は、あなたの鎧を腐食させる事ができます！　もしプレーヤが金属製
の鎧を着ている時にそのような怪物が攻撃してくると、その鎧は防御力がいくらか失
われてしまうでしょう。これと同じ腐食性の性質は、プレーヤが怪物を攻撃する時に
もその効果を持ちます。「保護」のスクロールを探してあなたの鎧や武器を腐食から
ガードしてください。

プレーヤは洞窟でたくさんのポーションやスクロールを見つけるでしょう。スクロー
ルを読んだりポーションを飲んだりすると、たいていは何か不思議な出来事が起こる
でしょう。ポーションとスクロールは、呪われているかもしれませんし祝福されてい
るかもしれません。このXRogueのバージョンでは、Monster Confusion(怪物混乱)の
スクロールはあなたの手を様々な色に変えるでしょう。祝福されたMagic Mapping(魔
法地図)のスクロールは非常に詳細な地図を示してくれます。Genocide(大量虐殺)の
スクロールは、洞窟の中だけでなく、外の世界にも同様の効果を持ちます。祝福され
たTeleportation(テレポート)のスクロールは、2～3階上へテレポートさせてくれま
す。祝福されたRemove Curse(呪い解除)は、怪物の近くで読むと、ある種の怪物をパ
ニックに陥れる原因にもなるでしょう。Charm Monster(魅力怪物)は、あなたにいく
つかの怪物を魅了させます。

プレーヤは、最大8つの指輪をする事ができます。いくつかの指輪は、それをしてい
る間中、プレーヤに不思議な力を与えます。いくつかの指輪は、プレーヤの代謝を促
進させるため、プレーヤは普段よりも速い頻度で食事をする必要があります。指輪は
呪われていたり祝福されていたりします。呪われた指輪は、呪い解除のスクロールを
読むか魔法をかけるかしないと、プレーヤは外す事ができません。

魔法の杖、ロッド、棒などは、戦闘中プレーヤを助けたり洞窟内で作用したりしま
す。プレーヤは怪物に攻撃したりテレポートさせたり、あるいは暗い部屋を明るくし
たりするために、杖を使うコマンド「z」(zap(撃つ))を使います。杖は呪われていた
り祝福されていたりします。

プレーヤは食料を倹約しなければなりません。洞窟内で移動したり創作したり、怪物
と戦ったりすると、エネルギーを消費します。飢えていくと、プレーヤの空腹により
気絶している期間はますます長くなり、その間、近くにいる怪物がプレーヤを攻撃し
放題となります。食料は、標準的な食料の包みであるか、様々な果実の形をしていま
す。いくつかの果実は、上を満たす事に加えて、副作用も持っています。
Slime-Molds(スライムモールド)は怪物の食料であり、あなたがそれを持っていて、
怪物が「友好的」な種類であったなら、あなたは標準的な食料の包みと交換してもら
えるかもしれません。

洞窟内で、金塊にはいくつかの用途があります。最初の用途は、Trading Post(交易
場)で、またはQuartermaster(補給係将校)からものを買う事です。Trading Post(交
易場)が見つかったら、階段を下りるのと同じ「>」コマンドで入る事ができます。
Quartermaster(「q」、補給係将校)は、ある特定の時間にだけ現れる洞窟内の行商人
で、彼の持っている商品をプレーヤに売りつけようとします。Quartermaster(補給係
将校)の商品は絶対に呪われてはいませんし、祝福されていたりする事もあります
が、普通の商品よりも値段が高めです。もしプレーヤがQuartermaster(補給係将校)
の申し出た何かを買うと決めたら、かれは所定の量の金塊と取引をして、姿を消しま
す。Quartermaster(補給係将校)を攻撃すると、急に消え失せます！　あなたは時々
「魔法のプール」の底で金塊を見つける事があります。潜って金塊をとるために
「>」コマンドが使えますが、溺れないように注意してください！

新しくゲームを始める時、プレーヤは選んだキャラクタのタイプに基づいた金塊の配
当を持って、Trading Post(交易場)からスタートします。キャラクタによって、ある
特定のアイテムの使用に関して、若干の制限があります。例えば、Fighters(ファイ
ター)、Paladins(聖騎士)、そしてRangers(レンジャー)だけがtwo-handed swords(両
手用の剣)を装備できるのに対して、Thieves(盗賊)やAssassins(刺客)はある特定の
種類の鎧の類を着る事ができません。しかし、Trading Post(交易場)(と
Quartermaster(補給係将校))は、あなたがそれを使えるかどうかにかかわらず、何で
もプレーヤに首尾良く売りつけます。

Pair of Boots(1対のブーツ)やBook(本)のような雑多な魔法のアイテムは、洞窟の中
に非常にたくさんあります。これらのアイテムは通常プレーヤの利益となりますし、
当然呪われてはいないと思われます。Cloak(外套)のようないくつかのアイテムは着
られますし、他のBook(本)やBeaker(ビーカー)、Ointment(軟膏)などは使用されるこ
とになっています。Medicine Crystal(医療用水晶)はあなたを癒し、総ての呪いを取
り除き、更にあなたを癒し、いくつかの怪物たちをパニックに陥れるかもしれませ
ん。

きわめて特別な魔法のアイテムやアーティファクトを持ったたくさんのユニークな怪
物が、洞窟の奥深くに存在しています。あなたがゲームを開始すると、あなたは洞窟
から取り戻す「クエストアイテム」を選ぶように言われます。もしその人のゲームに
おいて個人的なクエストアイテムではなかったとしても、これらのアイテムの多くは
プレーヤの利益として使う事ができます。しかし、それらのいくつかは知性を持ち、
手荒く扱われたり虐待されたりする事を拒絶したりするので、取り扱いには十分注意
しなければなりません。これらのアイテムは、運んでいると食料の使用が激しくなる
という点で、あなたの食料(そしてあなたの金塊)を消費し尽くします。あなたが何か
のアーティファクトを武器として装備している際、別のアーティファクトを拾い上げ
ようとすると、いくつかのアーティファクトアイテムは完全にあなたを殺してしまう
でしょう。アーティファクトについて、あまり強欲になってはいけない！　クエスト
アイテムは、XRogueの40～50階の間に現れ始めます。あまりトラウマ的とはならない
クエストアイテムは速く現れ始めるのに対して、もっとシビアなものは遅くにしか現
れません。いったんアーティファクトを持つ唯一の怪物が殺されると、洞窟内であな
たが同じアーティファクトを持った別の怪物に出会う事はありません。

洞窟には、trap doors(次の階へ落とされる罠のドア)、bear traps(ネズミ捕り)、そ
してsleeping traps(催眠ガスの罠)など、様々な罠が仕掛けられています。時々、罠
は怪物やプレーヤがその上を通るまで、見えないように隠されています。いったん使
われた罠は、見えるようにはなりますが、機能し続けます。賢い怪物はそれを踏まな
いようにしています。ワームホールと呼ばれる罠は、あなたを洞窟の「外の領域」へ
転送します。そこでは、あなたは自分の元居た場所へ帰る方法を見つけ、速やかにそ
こへ帰るまで、見知らぬ恐竜と戦う事になります。

Quartermaster(補給係将校)以外の各々の怪物は、洞窟の限られた階の範囲にしか現
れません。同じタイプの総ての怪物は同じ能力を持っています。例えば、総ての
giant rats(巨大ネズミ)はプレーヤを病気にしますし、総てのjackalweres(ジャッカ
ルワー)はプレーヤを眠らせます。しかし、同じタイプの怪物であっても、
strength(力)やintelligence(知性)は異なります。例えば、ある1匹のkobold(コボル
ド)は、別のコボルドよりも、殺すのが非常に難しいかもしれません。一般に、殺す
のが難しい怪物ほど、多くの経験値の価値があります。

ほとんどの怪物たちはかみついたり爪でひっかいたりして攻撃しますが、中には武器
を持っていたり自らの生命力を武器として使ってくるものがいます。いくつかの怪物
たちは、杖やアーティファクトなどの魔法のアイテムを使う事さえできるのです。飛
び道具を持っていたり魔法が使えたりする怪物たちは、部屋越しに、あるいは通路沿
いにプレーヤを攻撃してくることができます。

いくつかの怪物たちは他のものよりも知的で、知的な怪物ほど、死にそうになった時
に逃げます。追い詰められた状況でない限り、逃げている怪物はプレーヤを攻撃して
きません。

時には、怪物からの援助を受けることができます。例えば、charm monster(怪物魅
惑)のスクロールを読んだり、charm monster(怪物魅惑)の聖歌を歌ったりすると、怪
物にプレーヤが友達であると思わせる事ができます。魅了された怪物は、自分と同じ
種族の怪物でない限り、プレーヤが的とする怪物に対して、一緒に戦うでしょう。次
の階へ行く時、あなたに魅了された怪物があなたと同じ部屋にいる事を確かめてくだ
さい。もしいないければ、彼らはもうどこかへ行ってしまいます。

オプション:

このゲームにはいくつかのオプションがあります。いくつかはゲームの最初に設定で
きるだけのもので、残りはゲーム中にも設定できます。

      default - 「デフォルト」属性設定のキャラクタでゲームをします。
        terse - スクリーン最上行に、簡略メッセージで表示します。
         jump - プレーヤの移動表示をスピードアップします。
         step - 一度に総ての持ち物をリストアップします。
      overlay - 持ち物リスト表示を、現在のスクリーンにオーバーレイ表示しま
                す。
        flush - 前に入力したキャラクタをフラッシュします。
        askme - 新しいタイプのスクロールやポーションに名前を付けるプロンプト
                を出します。
       pickup - アイテムの上に乗ると、自動的にそれを拾い上げます。
         name - プレーヤの名前です。
         file - ゲームをセーブするファイル名です。デフォルトはxrogue.savで
                す。
        score - ゲームのスコアファイルの場所を特定します。
         type - キャラクタのタイプを特定します(変更できません)。
 quested item - クエストアイテム。ゲーム開始時に設定します(変更できません)。

プレーヤは、ROGUEOPTS環境変数を通じて、ゲームの初めにゲームオプションをセッ
トする事ができます。

若干の例:

  ROGUEOPTS="default nopickup, nooverlay, name=Corwin, class=magician"
  ROGUEOPTS="pickup, overlay, file=xrg.sav, score=/home/games/scorefile"

プレーヤは、「o」(オプション)命令を通じて、ゲーム中いつでもオプション設定を
変える事ができます。オプションメニューで、新しい値を入力すると設定が変わり、
Enter(RETURN)でリストの次のオプションへ移動します。「-」を入力すると前のオプ
ションへ戻り、ESCAPEで通常のゲーム画面に戻ります。

XRogueでは新しく「default」(デフォルト)というオプションが追加されました。あ
なたがROGUEOPTS環境変数に「default」という単語を入れると、あなたのキャラクタ
は属性設定、鎧、武器、クエストアイテム、そして食料総てが「default」設定にな
ります。

「default」を設定しておくと、あなたの属性ポイントを選択/分配する最初の画面を
強制的に飛ばします。また、鎧、武器、クエストアイテムなどを選択する最初の「装
備ページ画面」さえも飛ばしてしまいます。

もしあなたがROGUEOPTSで「default」と一緒に「class」オプションも設定しておく

異なるクエストアイテムを選択できるのは、最初にTrading Post(交易場)にいるとき
だけです(もしあなたがROGUEOPTSを通じて「default」設定をしていなければ)。

プレーヤが殺されると、スコアは経験値と金塊(10％減らされます)の合計値と等しく
なります。プレーヤはゲームを止める事で、金塊が10％減らされるのを防ぐ事ができ
ます。もしプレーヤが生きて洞窟から地上へと戻れたならば、経験値と金塊に加え
て、荷物に入っているアイテムを総て売り払った金も追加したものが彼のスコアとな
ります。

このゲームはトップ20のプレーヤをスコアファイルとして保存していきます。インス
トールオプションとして、ゲームがキャラクタタイプと名前毎に3つまでのエントリ
しか記録しないようにコンパイルされているかもしれません。
