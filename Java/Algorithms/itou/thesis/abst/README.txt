修士論文・卒業論文 スタイルファイル


修士論文, 卒業論文に関連した LaTeX2ε用のスタイルファイルを用意しました.

クラスファイル, スタイルファイル
  AIabst.cls:       概要用クラスファイル
  AIcover.cls:      表紙用クラスファイル
  AIthesis.sty:     論文用スタイルファイル(参考)
  personal.sty:     個人情報用スタイルファイル

サンプルファイル
  abst.tex:         概要サンプル(兼 概要, 表紙ファイルの使い方)
  guide.tex:        論文サンプル(兼 論文書式について)
  personal.tex:     個人データサンプル
  cover1.tex:       論文表紙
  cover2.tex:       論文中表紙
  spine.tex:        論文背表紙

ドキュメント
  abst.pdf:         サンプル(abst.tex)の PDF 版
  guide.pdf:        サンプル(guide.tex)の PDF 版
  Bschedule.pdf:    卒業論文スケジュール
  Mschedule.pdf:    修士論文スケジュール
  
アーカイブ(上記ファイルをまとめたもの)
  thesis.tar.gz:    EUC 版
  thesis.zip:       S-JIS 版
