     Ohio Scientific Model 600/Superboard II Hardware Memory Map
                   Jeff Tranter <tranter@pobox.com>
                             02 May 2023

+---------------+---------------------------------------------------+
| Address Range | Description                                       |
+---------------+---------------------------------------------------+
| $0000 - $00FF | Zero page RAM                                     |
| $0100 - $01FF | Stack RAM                                         |
| $0000 - $1FFF | RAM (8K)                                          |
| $A000 - $A7FF | ROM (Basic 1)                                     |
| $A800 - $AFFF | ROM (Basic 2)                                     |
| $B000 - $B7FF | ROM (Basic 3)                                     |
| $B800 - $BFFF | ROM (Basic 4)                                     |
| $D000 - $D3FF | Video RAM                                         |
| $D085 - $D37C | Visible video RAM (24x24 mode)                    |
| $D400 - $D7FF | Color RAM (bits D0-D3 only)                       |
| $D800         | Sound/Bank/Color/Video register (Decimal 55296)   |
| $DF00         | Keyboard register (Decimal 57088)                 |
| $F000         | 6850 ACIA Status/Control Register (Decimal 61440) |
| $F001         | 6850 ACIA Data Register (Decimal 61441)           |
| $F800 - $FFFF | SYN600 ROM (IO routines and monitor)              |
| $FFFA - $FFFB | NMI vector                                        |
| $FFFC - $FFFD | Reset vector                                      |
| $FFFE - $FFFF | IRQ vector                                        |

$F000 61440 6850 ACIA Status/Control Register
+----+----+----+----+----+----+----+----+-------------------------+
| D7 | D6 | D5 | D4 | D3 | D2 | D1 | D0 | Description             |
+----+----+----+----+----+----+----+----+-------------------------+
|/IRQ| PE |OVRN| FE |/CTS|/DCD|TDRE|RDRF| Status Register (read)  |
+----+----+----+----+----+----+----+----+-------------------------+
| CR7| CR6| CR5| CR4| CR3| CR2| CR1| CR0| Control Register (write)|
+----+----+----+----+----+----+----+----+-------------------------+

+---+---+--------------+
|CR1|CR0| Function     |
+---+---+--------------+
| 0 | 0 | Divide by 1  |
| 0 | 1 | Divide by 16 |
| 1 | 0 | Divide by 64 |
| 1 | 1 | Master reset |
+---+---+--------------+

+---+---+---+----------+
|CR4|CR3|CR2| Function |
+---+---+---+----------+
| 0 | 0 | 0 | 7E2      |
| 0 | 0 | 1 | 7O2      |
| 0 | 1 | 0 | 7E1      |
| 0 | 1 | 1 | 7O1      |
| 1 | 0 | 0 | 8N2      |
| 1 | 0 | 1 | 8N1      |
| 1 | 1 | 0 | 8E1      |
| 1 | 1 | 1 | 8O1      |
+---+---+---+----------+

+---+---+-----------------------------+
|CR6|CR5| Function                    |
+---+---+-----------------------------+
| 0 | 0 | /RTS low, TX int. disabled  |
| 0 | 1 | /RTS low, TX int. enabled   |
| 1 | 0 | /RTS high, TX int. disabled |
| 1 | 1 | /RTS low, transmit break    |
+---+---+-----------------------------+

+---+--------------------+
|CR7| Function           |
+---+--------------------+
| 0 | Disable interrupts |
| 1 | Enable interrupts  |
+---+--------------------+

| D7 | D6 | D5 | D4 | D3 | D2 | D1 | D0 | Receive Data (read)   |
+----+----+----+----+----+----+----+----+-----------------------+
| D7 | D6 | D5 | D4 | D3 | D2 | D1 | D0 | Transmit Data (write) |
